# pchelna-magia
Пчелна Магия е интерактивна образователна игра, създадена да превърне ученето на таблицата за умножение в забавно приключение. Играчът помага на работлива пчеличка да събере прашец, като решава математически задачи. С визуално стимулиращ дизайн, анимации и звукови ефекти, играта е идеален помощник за деца в началното училище.
